# Radio 单选框

通过鼠标在一组备选项中进行单选

# 基础用法

<template>
  <div>
    <mc-radio v-model="radio" label="1">1</mc-radio>
    <mc-radio v-model="radio" label="2">2</mc-radio>
  </div>
</template>
<script>
export default {
  data() {
    return {
      radio: "1",
    };
  },
};
</script>
